## this is a clojure-friendly emacs config

If you're new to emacs, check out:

- [basic emacs](http://www.braveclojure.com/basic-emacs/)
- [emacs tutorial](http://clojure-doc.org/articles/tutorials/emacs.html)

## useful packages
 - magit (git)
 - helm (file, buffer quick switch)
 - helm swoop
 - projectile
 - company
 - org-mode
 - eldoc
 - rainbow-delimiters
 - which key
 - ace jump
 - tomorrow theme
 - multi-cursors
 - expand-region
 - neotree
 - yasnippet
 - whitespace cleanup

## for Clojure

 - clojure-mode
 - cider (compile, ut, debug)
 - paredit
 - smartparens (strengthen the paredit mode)

## for Erlang

 - erlang-mode
 - edts

# for Go

 - go-mode
 - go-autocomplete
 - go-eldoc
